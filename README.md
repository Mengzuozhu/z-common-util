# z-common-util
Java常用工具类封装

## 模块

- `common-util`
- `excel-util`
- `json-util`
- `web-util`

## 快速开始

1. 添加`common-util` Maven引用

   ```xml
           <dependency>
               <groupId>com.github.mengzuozhu</groupId>
               <artifactId>common-util</artifactId>
               <version>1.0-SNAPSHOT</version>
           </dependency>
   ```
